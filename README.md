# Affiliate Automator Digital Product Site

This example project provides a simple Node.js/Express app that lets you generate digital product content using the OpenAI API and list products for sale.

## Setup

1. Install dependencies

```bash
npm install
```

2. Create a `.env` file with your OpenAI API key:

```
OPENAI_API_KEY=your-key-here
```

3. Start the server

```bash
node server.js
```

Then open `http://localhost:3000` in your browser.

This is a minimal demonstration and does not include authentication or payment processing.
